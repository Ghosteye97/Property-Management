You are a senior full-stack software architect and product designer.

Build a fully functional, production-ready Property Management Application designed for property managers handling:
- Sectional Title schemes
- Homeowners Associations (HOAs)
- Residential and commercial complexes

The application must be modular, scalable, and user-friendly, with a modern SaaS architecture.

---

## 🏗️ TECH STACK REQUIREMENTS

Frontend:
- React (Next.js preferred)
- TailwindCSS (clean dashboard UI)
- Component-based architecture

Backend:
- Node.js (Express or NestJS)
- REST API (or GraphQL if appropriate)

Database:
- PostgreSQL (normalized schema)
- Prisma ORM (or equivalent)

Auth:
- JWT authentication with role-based access
- Roles: Admin, Property Manager, Finance, Support, Owner

Hosting-ready:
- Environment configs
- Docker support (optional but preferred)

---

## 🧩 CORE MODULES

### 1. COMPLEX / PROPERTY MANAGEMENT
- Create a new Complex:
  - Complex Name
  - Type (Sectional Title / HOA / Other)
  - Registration Number
  - Address
  - Number of Units
  - Managing Agent Details

- On creation:
  - Redirect to a FULL PAGE COMPLEX DASHBOARD VIEW

---

### 2. COMPLEX DASHBOARD (MAIN VIEW)

This is the core UI/UX experience.

Layout should include:

LEFT SIDEBAR:
- Overview
- Units
- Billing
- Maintenance / Support
- Documents
- Communication (Newsletters)
- Reports
- Settings

MAIN PANEL:
- Summary cards:
  - Total Units
  - Occupied / Vacant
  - Outstanding Balances
  - Open Maintenance Requests

---

### 3. UNIT MANAGEMENT

- Auto-generate units based on number entered
- Ability to:
  - Edit units individually
  - Bulk edit units
  - Assign owners / tenants
  - Track unit status

Each Unit should have:
- Owner details
- Tenant details
- Billing profile
- Maintenance history
- Documents

---

### 4. BILLING & FINANCE

- Generate levy invoices
- Bulk billing runs
- Upload / manage billing files
- Track payments and outstanding balances

Integrations:
- QuickBooks Online (QBO)
- Pastel Partner (simulate API if needed)

Features:
- Export reports (CSV/PDF)
- Payment reconciliation

---

### 5. MAINTENANCE / SUPPORT

- Ticketing system:
  - Log maintenance requests per unit
  - Assign to staff/vendors
  - Track status (Open, In Progress, Completed)

- Attach images/files
- Notifications

---

### 6. COMMUNICATION

- Send newsletters / notices to:
  - Entire complex
  - Selected units

- Email templates
- Communication history log

---

### 7. DOCUMENT MANAGEMENT

- Upload and manage:
  - Compliance documents
  - Contracts
  - Financial statements

- Per complex and per unit storage

---

### 8. REPORTING & DASHBOARDS

- Financial reports
- Occupancy reports
- Maintenance reports

- Visual dashboards (charts/graphs)

---
